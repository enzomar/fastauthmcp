Enterprise Authentication
for FastMCP

Ceramic is a drop-in framework that adds authentication, authorization, observability and enterprise security to any FastMCP application — without touching a single tool.

Get Started → Star on GitHub
$ pip install ceramic-fwk

the change

Two lines. That's it.

Point Ceramic at your existing FastMCP app and every tool call is authenticated, authorized and observed — with zero changes to your tools.

weather_server.py 
# before
from fastmcp import FastMCP

app = FastMCP("Weather")
weather_server.py ceramic 
# after
from ceramic import FastMCP

app = FastMCP(
    "Weather",
    config="ceramic.yaml"
)

Security-first defaults

          
PKCE, TLS 1.2+, platform-native token storage, and automatic log redaction — all enabled by default.

        

        

          

            
          

          
Zero lock-in

          
Remove Ceramic and your FastMCP app still works. It's a layer on top, not a fork underneath.

        

        

          

            
          

          
Production-tested patterns

          
Middleware pipeline, context propagation, and hot reload follow battle-tested patterns from Django, FastAPI, and Express.
